## Yahoo! Japan Weather

Unofficial Home Assistant integration for Yahoo!天気・災害.

- Guided selection by prefecture, forecast area, and municipality
- Three-hour and up to eight-day forecasts
- No API key required
- English, Japanese, and Simplified Chinese UI
- Atomic location reconfiguration with stale-page and URL validation
- Privacy-safe Home Assistant diagnostics

See the repository README for installation, setup, data details, and limitations.
